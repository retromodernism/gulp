Чтобы перенести сборку, нужно в папку проекта перенести файлы 
	#src
	gulpfile.js
	package.json
Далее в терминале прописать npm i

	Доп. команды:
		gulp svgSprite - объеденить иконки в один файл
		gulp otf2ttf - конвертировать otf в ttf
		
Что включает сборка:
	препроцессор SCSS
	автоматическая сборка в папку для экспорта
	работа со шрифтами
	преобразование svg иконок в один файл
	минификация css, js кода
